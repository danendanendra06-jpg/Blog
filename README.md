# 📝 Personal Blog

Welcome to my Blog repository! This is a personal blog website built from scratch using core web technologies (**HTML, CSS, and JavaScript**). It serves as a space where I share my thoughts, learning notes, project documentations, and personal stories.

---

## 🚀 Key Features
* **Responsive Design:** Optimized for a seamless reading experience across mobile, tablet, and desktop devices.
* **Clean Structure:** Straightforward navigation to easily browse through different articles.
* **Lightweight & Fast:** Built using vanilla web technologies without any heavy frameworks.

---

## 📂 Folder Structure
Here is a quick overview of the project's layout:
* `css/` : Contains stylesheets for the blog's layout and design.
* `js/` : Contains JavaScript pieces for page interactivity.
* `pages/` : The directory where blog articles and other sub-pages are stored.
* `index.html` : The main entry point (homepage) of the blog.
* `Blog-ethic.txt` : Articles and ethical standards for writing content.

---

## 🛠️ Tech Stack
* **HTML5** - For content structuring.
* **CSS3** - For styling and responsive layouts.
* **JavaScript (ES6+)** - For logic and simple interactive elements.

---

## ✍️ Contribution & Ethics
To maintain the quality of the content, the writing process follows the guidelines specified in `Blog-ethic.txt`. If you spot any bugs or have suggestions for visual improvements, feel free to open an **Issue** or submit a **Pull Request**.
